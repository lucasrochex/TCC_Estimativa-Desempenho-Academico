# TCC_Estimativa-Desempenho-Academico
Contém Trabalho de Conclusão de Curso do curso Ciência de Dados e Big Data.

Nome do Trabalho: Estimativa do Desemepenho Acadêmico Utilizando Técnicas de Aprendizado de Máquina

## Descrição dos arquivos presentes:
**Notebooks:**

Análise Exploratória de Dados.ipynb - Notebook referenciado no trabalho  
Clusterização dos alunos.ipynb - Notebook referenciado no trabalho  
Informação mútua e Correlação.ipynb - Notebook referenciado no trabalho  
Modelos de Machine Learning para Previsão das Notas - Notebook referenciado no trabalho.

**Datasets básicos**

student-por.csv - Notas em português e informações sobre os alunos  
student-mat.csv - Notas em matemática e informações sobre os alunos  

**Arquivos intermediários**

Matematica - Atibutos codificados.xlsx - Notas de matematica após passarem por "Label Encoding"  
Matematica - Resultados de Informacao Mutua.xlsx - Informação Mútua de cada atributo com as notas de matemática  
Math modificada com clusters.xlsx - Dataset básico das notas de matemática com o acréscimo dos clusteres nos quais cada aluno foi classificado.  


**Relatório Final**:

Estimativa do Desempenho Acadêmico com Técnicas de Aprendizado de Máquina_Lucas Rocha Rodrigues.pdf
